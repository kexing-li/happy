## 数据库设计

### 表结构

1. **profiles** - 用户信息扩展
   - id (UUID, PK, FK -> auth.users)
   - display_name (VARCHAR)
   - is_admin (BOOLEAN)

2. **game_records** - 麻将记录（核心表）
   - id (SERIAL, PK)
   - sequence_no (INT) - 序号
   - year (INT) - 年份
   - game_date (VARCHAR) - 日期
   - player_zhou/xing/fei/jun/ping (INT) - 五位团员盈亏
   - rake (INT) - 抽水
   - table_fee (INT) - 台费
   - meal_cost (INT) - 吃饭
   - other_cost (INT) - 其他开销
   - daily_remaining (INT) - 当日剩余水费
   - fund_balance (INT) - 剩余团费
   - remarks (TEXT) - 备注
   - is_summary (BOOLEAN) - 是否为汇总行

3. **upload_logs** - 上传日志
   - id (SERIAL, PK)
   - user_id (UUID, FK)
   - filename (VARCHAR)
   - row_count (INT)
   - uploaded_at (TIMESTAMPTZ)

### 视图

- **stats_overview** - 总览统计
- **stats_players** - 团员统计
- **stats_yearly** - 年度统计

## 上传流程

1. 管理员选择 Excel 文件
2. 前端通过 FormData 发送到 Server Action
3. 服务端使用 xlsx 库解析 Excel
4. 清洗数据（跳过表头、处理空值、识别汇总行）
5. 事务中：删除旧数据 → 插入新数据 → 记录日志
6. 返回处理结果
