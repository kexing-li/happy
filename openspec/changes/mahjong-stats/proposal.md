# 麻将团费统计系统

## 概述

为麻将团提供一个数据统计和可视化系统，管理员可上传 Excel 记录文件，所有团员登录后可查看统计数据。

## 背景

团队定期聚会打麻将，需要记录每场的盈亏、抽水、吃饭等费用，并统计团费结余。当前使用 Excel 文件手工记录，需要一个 Web 界面来展示这些数据。

## 目标

1. **管理员上传**：管理员可上传 Excel 文件，系统自动解析并存入数据库
2. **数据可视化**：提供仪表盘展示总览统计、团员排行、趋势图等
3. **详细记录**：支持分页、筛选、搜索查看历史记录
4. **权限控制**：只有登录用户可查看，只有管理员可上传

## 技术方案

- **前端**：Next.js 16 + React 19 + Tailwind CSS 4
- **后端**：Supabase (PostgreSQL + Auth + RLS)
- **Excel 解析**：xlsx 库
- **图表**：recharts

## 数据结构

Excel 文件包含以下字段：
- 基础信息：序号、年份、日期
- 团员盈亏：周、星、飞、俊、平（5人固定）
- 开销：抽水、台费、吃饭、其他
- 结余：当日剩余水费、剩余团费
- 备注

## 非目标

- 不支持在线编辑记录（只能通过上传 Excel 更新）
- 不支持多团队（单一团队使用）
- 暂不做移动端适配优化

## 风险

- Excel 格式变化可能导致解析失败，需要做好错误处理
